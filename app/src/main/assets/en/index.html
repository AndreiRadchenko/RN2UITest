Work beginning

The application is designed to store and send USSD and SMS commands. The main screen contains two tabs for USSD and SMS codes, respectively. Clicking on a list item results in the transfer of the code to the dialer, or the program for sending SMS messages. Your can add codes through importing codes , adding your own codes , or editing codes with saving a copy.

Importing codes

To import mobile operators USSD codes or SMS banking commands, open the sidebar and click import. You will be offered a list with the code sets for download. The list includes only those banks that offer their customers management services using SMS commands. When you click on a list item, USSD codes are downloaded to the USSD tab of the application, SMS banking commands to the SMS tab.

Adding your own codes

To add your own USSD code, go to the USSD tab and click the floating button in the lower right corner of the screen. Complete the USSD editing form. The "code" field must contain the USSD code. Click the button to save the code to the list.

Editing codes

In order to change the code or its name, click on the icon , located in the right corner of each item of the list. Make changes in the editing window. Click the button to save the changes, or click the button to save the copy as a new code.

Deleting and sorting codes

To delete a list item, press and hold it. You will be in the selection mode. Select other items by clicking on them, or select all items by clicking the button . To delete the selected codes, click on the icon .

You can drag items up and down by holding the icon or the code icon.

Creating a backup

To save a backup of the codes set, open the sidebar and click the backup. If desired, add a comment in the dialog box and click OK. The backup will be stored in the external memory of the phone.

To restore a set of codes from backup, open the sidebar and click the item restore. In the window that appears, select the backup to be restored and click on it. The restored codes are added to the end of the list in the corresponding tabs - USSD codes in the USSD tab, SMS commands - in the SMS tab.

Protection with a pin code or fingerprint

The application can contain codes that control your mobile or bank accounts. To protect against unauthorized access, use the PIN lock or fingerprint lock.

To enable the security feature, open the sidebar and select the settings. Set a pin code and fingerprint, if such function is supported by your phone.
