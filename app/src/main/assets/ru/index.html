Начало работы

Приложение предназначено для хранения и отправки USSD и SMS команд. Главный экран содержит две вкладки для USSD и SMS кодов соответственно. Клик по элементу списка приводит к передаче кода в номеронабиратель, или программу отправки SMS сообщений. Добавить коды можно через импорт кодов , добавление собственных кодов , или редактирование кодов с сохранением копии.

Импорт кодов

Для импорта USSD кодов операторов связи или SMS команд мобильного банкинга, откройте боковую панель и нажмите импорт кодов. Вам будет предложен список с наборами кодов для загрузки. В списке представлены только те банки, которые предлагают своим клиентам управления услугами с помощью SMS команд. При нажатии на элемент списка, USSD коды загружаются во вкладку USSD приложения, команды SMS банкинга - во вкладку SMS.

Добавление собственных кодов

Чтобы добавить собственный код USSD, перейдите на вкладку USSD и кликните по всплывающей кнопке в правом нижнем углу экрана. Заполните форму редактирования USSD. Поле "код" должно содержать USSD код. Нажмите кнопку для сохранения кода в список.

Чтобы добавить SMS команду, перейдите на вкладку SMS и кликните на кнопку . Заполните форму редактирования SMS. Поле "код" должно содержать SMS команду, поле "номер телефона" - номер, на который будет отправлена команда. Для сохранения кода нажмите

Редактирование кодов

Для того, чтобы, изменить код или его название кликните по иконке , расположенной в правом углу на каждом элементе списка. В окне редактирования сделайте правки. Нажмите на кнопку для сохранения изменений, или на кнопку для сохранения копии как нового кода.

Удаление и сортировка кодов

Чтобы удалить элемент списка, нажмите и удерживайте его. Вы перейдете в режим выделения. Если необходимо, выделите другие элементы, кликая по ним, или выделите все элементы нажав кнопку . Для удаления выбранных кодов кликните по иконке .

Элементы списка можно перетаскивать вверх и вниз удерживая за значок или за иконку кода.

Создание резервной копии

Для сохранения резервной копии набора кодов, откройте боковую панель и кликните пункт резервная копия. По желанию, добавьте комментарий в диалоговом окне и нажмите ОК. Резервная копия будет сохранена во внешней памяти телефона.

Чтобы восстановить набор кодов из резервной копии, откройте боковую панель и кликните пункт восстановить копию. В появившемся окне выберите резервную копию для восстановления и кликните по ней. Восстановленные коды добавляются в конец списка в соответствующих вкладках - USSD коды во вкладку USSD, SMS команды - во вкладку SMS.

Защита пин-кодом или отпечатком пальца

Приложение может содержать коды управляющие вашими мобильными или банковскими счетами. Для защиты от несанкционированного доступа воспользуйтесь блокировкой входа пин-кодом или отпечатком пальца

Чтобы включить функцию защиты, откройте боковую панель и выберите настройки. Установите пин-код и разблокирование по отпечатку пальца, если такая функция поддерживается вашим телефоном.
